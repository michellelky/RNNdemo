# RNNdemo

An experiment with wix's [react-native-navigation](https://wix.github.io/react-native-navigation/#/)
Currently supports ios 

## Navigation Features

* Tab-based 
* Push screen
* Drawer

## Built With

* React Native
* TypeScript

## Installation
```
$ git clone git@github.com:michellelky/RNNdemo.git
$ cd RNNdemo
$ yarn install
```

## Running the App
```
$ react-native run-ios
```
